## ЁЯЪА How to run

1. Copy `.env.example` into `.env`, otherwise ENV validation will fail and throw an Error.
```sh
cp .env.example .env
```
2. Install dependencies.
```sh
pnpm i
```
3. Run dev server.
```sh
pnpm dev
```
